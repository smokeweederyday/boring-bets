# Stadium curation guide\n\nThis file defines the image-art direction for MLB game-header stadium hero shots.\n\n## Global standard\n\n- Use the Yankee Stadium image as the composition reference.\n- Favor a view from near home plate looking out toward the field and upper sky.\n- Outdoor parks should feel like roughly half sky and half stadium.\n- Fixed-roof parks should show the closed dome/roof clearly.\n- Retractable-roof parks should have open and closed variants.\n- When a retractable roof is open, the outdoor time of day should be reflected.\n\n## Local file naming\n\n- Default: `VENUEID.webp`\n- Outdoor time-of-day variants: `VENUEID-day.webp`, `VENUEID-dusk.webp`, `VENUEID-night.webp`\n- Retractable open variants: `VENUEID-open-day.webp`, `VENUEID-open-dusk.webp`, `VENUEID-open-night.webp`\n- Retractable closed variant: `VENUEID-closed.webp`\n- Fixed-roof indoor variant: `VENUEID-closed.webp`\n\n## Current venue plan\n\n- Yankee Stadium (3313): outdoor -> day,
