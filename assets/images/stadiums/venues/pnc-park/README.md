# PNC Park

Managed admin image folder for PNC Park.

Current variants:

- `fair-day.webp`
- `fair-dusk.webp`
- `fair-night.webp`

There are intentionally no PNC Park rain or rain-delay photographs yet.
Ordinary rain falls back to the correct fair-weather day/night photograph.

Replace any image using the same filename. No JavaScript change is required.

Recommended export size: 1916 × 821 pixels.
